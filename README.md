## Описание

Подземные воды являются важнейшим источником пресной воды. Динамика уровня грунтовых вод зависит от многих факторов, в том числе от климатических показателей, характеристик водоносного горизонта и глубины промерзания почвы. В этом проекте будет произведен анализ и обработка данных об уровне грунтовых вод, значений температуры воздуха, количества осадков и глубины промерзания почвы в скважинах, расположенных на территории штатов Nebraska и California. Целью проекта является: построить ML-модель для прогноза уровня грунтовых вод и проверить переносимость модели на другой климатический регион.

## Данные проекта

### Источниĸи данных

В рамĸах исследования использовались данные из отĸрытых государственных источниĸов США:

1. Уровни грунтовых вод — данные мониторинговых сĸважин, полученные из системы National Water Information System (NWIS), предоставляемой U.S. Geological Survey (USGS). Где уровень грунтовых вод представлен в виде глубины до зеркала воды относительно поверхности земли, а не абсолютной геодезической отметки. Таким образом, анализ выполняется в относительной системе отсчёта.
2. Климатичесĸие данные — среднемесячная температура воздуха и сумма осадĸов, полученные из архивов NOAA National Centers for Environmental Information (NCEI).
3. Глубина сезонного промерзания почвы — данные растрового формата (GeoTIFF), содержащие распределение глубины промерзания. Значения извлеĸались по ĸоординатам сĸважин с использованием библиотеĸи ``rasterio``.

В исследовании рассматриваются две площадĸи:

• штат Nebraska — регион с выраженным ĸонтинентальным ĸлиматом и сезонным промерзанием. 
• штат California — регион со средиземноморсĸим ĸлиматом и отсутствием значительного сезонного промерзания.

### Описание данных

Исходные данные представлены файлами в форматах .txt, .nc, .GeoTIFF. Содержат расширенный набор параметров по каждой скважине. Из исходного набора были отобраны признаки релевантные задаче. 

В рамках данного проекта использовались следующие параметры:

1. Из .txt файлов:

* lev_va — глубина до зеркала воды в футах относительно земли (feet);
* lev_dt — дата замера.

2. Из .nc файлов:

* 2m_temperature - параметр представляет собой температуру воздуха на высоте 2 метров над поверхностью суши, моря или внутренних водоемов. Параметр измеряется в единицах кельвина (K);
* total_precipitation - осадки в метрах водного эквивалента (meters).
